# FolderForge AI - Tree Editor Upgrade

Esta atualização transforma o app em um criador visual de árvores para projetos e arquiteturas.

Arquivos incluídos:

- `src/main.jsx`
- `src/styles.css`

## Como aplicar no Termux

Extraia este ZIP na raiz do repositório `folderforge-ai`, substituindo os arquivos existentes.

Depois rode:

```bash
git add src/main.jsx src/styles.css
git commit -m "Add visual project tree editor"
git push https://SEU_TOKEN@github.com/estreladigital482-collab/folderforge-ai.git main
```

## O que foi adicionado

- Editor visual de árvore.
- Adicionar pasta raiz.
- Adicionar arquivo raiz.
- Adicionar pasta ou arquivo dentro de uma pasta selecionada.
- Renomear nós.
- Excluir nós.
- Templates por tipo de projeto.
- Salvar estrutura no navegador com localStorage.
- Exportar ZIP baseado na árvore atual.
- Lista dos arquivos que serão exportados.
